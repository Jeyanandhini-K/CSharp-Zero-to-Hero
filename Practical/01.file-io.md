# 📁 File I/O in C#

---

## 🧠 What is File I/O?

**File I/O** (Input/Output) means reading from and writing to files on disk. Almost every real app does this — saving settings, logging, reading config, exporting data.

```csharp
// Read a file
string content = File.ReadAllText("notes.txt");

// Write a file
File.WriteAllText("output.txt", "Hello, File!");
```

> 💡 C# gives you two levels:
> - **`File` / `Directory`** — simple, one-line helpers ✅ (use these first)
> - **`StreamReader` / `StreamWriter`** — for large files or fine control

---

## ✍️ Writing Files

```csharp
// Write all at once (creates or overwrites)
File.WriteAllText("hello.txt", "Hello, World!");

// Write multiple lines
string[] lines = { "Line 1", "Line 2", "Line 3" };
File.WriteAllLines("lines.txt", lines);

// Append to existing file (doesn't overwrite)
File.AppendAllText("log.txt", $"[{DateTime.Now}] App started\n");

// Write bytes (for binary files)
byte[] data = { 0x48, 0x65, 0x6C, 0x6C, 0x6F };
File.WriteAllBytes("binary.dat", data);
```

---

## 📖 Reading Files

```csharp
// Read entire file as one string
string content = File.ReadAllText("hello.txt");
Console.WriteLine(content);

// Read all lines into an array
string[] lines = File.ReadAllLines("lines.txt");
foreach (string line in lines)
    Console.WriteLine(line);

// Read bytes
byte[] data = File.ReadAllBytes("binary.dat");

// ✅ Async versions (always prefer for large files)
string contentAsync = await File.ReadAllTextAsync("hello.txt");
string[] linesAsync  = await File.ReadAllLinesAsync("lines.txt");
```

## 🌊 StreamReader / StreamWriter — For Large Files

When files are huge, don't load everything into memory. Stream it line by line:

```csharp
// ✅ StreamWriter — write large files efficiently
using StreamWriter writer = new StreamWriter("bigfile.txt");
for (int i = 1; i <= 100_000; i++)
{
    await writer.WriteLineAsync($"Line {i}");
}
// Disposed automatically ✅

// ✅ StreamReader — read large files line by line
using StreamReader reader = new StreamReader("bigfile.txt");
string? line;
while ((line = await reader.ReadLineAsync()) != null)
{
    Console.WriteLine(line);
}
```

> 💡 `File.ReadAllText` loads the whole file into memory. `StreamReader` reads one line at a time — much better for big files.

## 📂 Working with Paths

```csharp
// ✅ Always use Path.Combine — works on Windows AND Linux/Mac
string path = Path.Combine("data", "users", "profile.txt");
// Windows: data\users\profile.txt
// Linux:   data/users/profile.txt

// Useful Path helpers
string full      = Path.GetFullPath("notes.txt");          // absolute path
string dir       = Path.GetDirectoryName(full);            // folder only
string fileName  = Path.GetFileName(full);                 // "notes.txt"
string nameOnly  = Path.GetFileNameWithoutExtension(full); // "notes"
string ext       = Path.GetExtension(full);                // ".txt"
string temp      = Path.GetTempFileName();                 // temp file path

Console.WriteLine(dir);       // C:\Users\Asha\Documents (or similar)
Console.WriteLine(fileName);  // notes.txt
Console.WriteLine(ext);       // .txt
```

## 🗂️ Checking & Managing Files

```csharp
// Check existence
bool exists = File.Exists("notes.txt");
bool dirExists = Directory.Exists("data");

// Copy, Move, Delete
File.Copy("source.txt", "backup.txt");             // copy
File.Copy("source.txt", "backup.txt", overwrite: true); // overwrite ok
File.Move("old.txt", "new.txt");                   // rename/move
File.Delete("unwanted.txt");                       // delete

// File info
FileInfo info = new FileInfo("notes.txt");
Console.WriteLine(info.Length);          // size in bytes
Console.WriteLine(info.CreationTime);    // when created
Console.WriteLine(info.LastWriteTime);   // last modified
```

## 📁 Working with Directories

```csharp
// Create folders
Directory.CreateDirectory("data/logs");   // creates all missing folders

// List files in a folder
string[] files = Directory.GetFiles("data");
string[] txtFiles = Directory.GetFiles("data", "*.txt");           // filter
string[] allFiles = Directory.GetFiles("data", "*", SearchOption.AllDirectories); // recursive

// List subfolders
string[] folders = Directory.GetDirectories("data");

// Delete folder
Directory.Delete("temp", recursive: true);   // recursive = delete contents too
```

## 🌍 Real-World Example: Simple Logger

```csharp
public class Logger
{
    private readonly string _logPath;

    public Logger(string logPath)
    {
        _logPath = logPath;
        Directory.CreateDirectory(Path.GetDirectoryName(logPath)!);
    }

    public async Task LogAsync(string level, string message)
    {
        string entry = $"[{DateTime.Now:yyyy-MM-dd HH:mm:ss}] [{level}] {message}";
        await File.AppendAllTextAsync(_logPath, entry + Environment.NewLine);
    }

    public async Task<string[]> GetLogsAsync()
    {
        if (!File.Exists(_logPath)) return Array.Empty<string>();
        return await File.ReadAllLinesAsync(_logPath);
    }
}

// Usage
var logger = new Logger("logs/app.log");
await logger.LogAsync("INFO",  "Application started");
await logger.LogAsync("ERROR", "Something went wrong");

string[] logs = await logger.GetLogsAsync();
foreach (string log in logs)
    Console.WriteLine(log);
```

## 🎤 Interview Q&A

**Q1: What is File I/O in C#?**
A: Reading and writing data to files on disk using classes in `System.IO`.

**Q2: What's the difference between `File.ReadAllText` and `StreamReader`?**
A: `File.ReadAllText` loads the whole file into memory — fine for small files. `StreamReader` reads line by line — use it for large files to avoid memory issues.

**Q3: How do you append to a file without overwriting it?**
A: Use `File.AppendAllText`.
```csharp
File.AppendAllText("log.txt", "New line\n");
```

**Q4: Why use `Path.Combine` instead of string concatenation?**
A: It handles path separators correctly for every OS. `"data" + "/" + "file.txt"` breaks on Windows; `Path.Combine("data", "file.txt")` always works.

**Q5: How do you read a file asynchronously?**
A: Use the async versions of File methods.
```csharp
string content = await File.ReadAllTextAsync("file.txt");
```

**Q6: How do you create a folder if it doesn't exist?**
A: `Directory.CreateDirectory` — it creates all missing folders in the path and doesn't throw if it already exists.

**Q7: How do you list all files in a folder including subfolders?**
A: Use `SearchOption.AllDirectories`.
```csharp
string[] files = Directory.GetFiles("data", "*", SearchOption.AllDirectories);
```

**Q8: What happens if you `File.WriteAllText` an existing file?**
A: It **overwrites** it completely. Use `File.AppendAllText` to add to it instead.

**Q9: How do you get just the filename from a full path?**
A: `Path.GetFileName(fullPath)` — returns `"notes.txt"`. Use `Path.GetFileNameWithoutExtension` to drop the extension.

**Q10: Should StreamReader/StreamWriter be in a `using` block?**
A: Always — they hold file handles that must be released.
```csharp
using StreamReader reader = new StreamReader("file.txt");
```

## 🚀 Key Takeaways

✅ `File.ReadAllText` / `WriteAllText` — simple one-liners for small files  
✅ `StreamReader` / `StreamWriter` — stream large files line by line  
✅ Always use `Path.Combine` — never hardcode path separators  
✅ `File.AppendAllText` — add to a file without overwriting  
✅ Use async versions (`ReadAllTextAsync`) in async methods  
✅ `Directory.CreateDirectory` — creates all missing folders safely  
✅ Always wrap streams in `using` — releases file handles  
✅ `File.Exists` / `Directory.Exists` before reading — avoid exceptions
