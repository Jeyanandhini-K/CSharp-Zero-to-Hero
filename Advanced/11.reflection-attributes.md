# 🔍 Reflection & Attributes in C#

---

## 🧠 What is Reflection?

**Reflection** lets your code **inspect itself at runtime** — read class names, method names, properties, and even call methods dynamically without knowing them at compile time.

```csharp
// Without reflection — you know the type at compile time
var person = new Person { Name = "Asha" };
Console.WriteLine(person.Name);   // ✅ direct access

// With reflection — inspect ANY object at runtime
Type type = person.GetType();
Console.WriteLine(type.Name);          // Person
Console.WriteLine(type.FullName);      // MyApp.Models.Person
Console.WriteLine(type.Assembly.GetName().Name);   // MyApp
```

> 💡 Think of reflection as a **mirror** — your code can look at itself and see its own structure.

## 🔬 Inspecting Types

```csharp
public class Person
{
    public string Name { get; set; } = "";
    public int    Age  { get; set; }
    private string _secret = "hidden";

    public void Greet() => Console.WriteLine($"Hello, I'm {Name}");
    private void Secret() => Console.WriteLine("Shh!");
}

Type type = typeof(Person);   // or person.GetType()

// Get properties
PropertyInfo[] props = type.GetProperties();
foreach (var prop in props)
    Console.WriteLine($"Property: {prop.Name} ({prop.PropertyType.Name})");
// Property: Name (String)
// Property: Age  (Int32)

// Get methods
MethodInfo[] methods = type.GetMethods();
foreach (var method in methods)
    Console.WriteLine($"Method: {method.Name}");

// Get ALL members including private
FieldInfo[] fields = type.GetFields(
    BindingFlags.NonPublic | BindingFlags.Instance);
foreach (var field in fields)
    Console.WriteLine($"Field: {field.Name}");   // _secret
```

## ⚡ Reading & Writing Properties Dynamically

```csharp
var person = new Person { Name = "Asha", Age = 30 };
Type type = person.GetType();

// Read property value dynamically
PropertyInfo nameProp = type.GetProperty("Name")!;
string? name = (string?)nameProp.GetValue(person);
Console.WriteLine(name);   // Asha

// Write property value dynamically
nameProp.SetValue(person, "Ravi");
Console.WriteLine(person.Name);   // Ravi

// Loop through all properties and print values
foreach (var prop in type.GetProperties())
{
    object? value = prop.GetValue(person);
    Console.WriteLine($"{prop.Name} = {value}");
}
// Name = Ravi
// Age  = 30
```

## 🚀 Calling Methods Dynamically

```csharp
var person = new Person { Name = "Asha" };
Type type = person.GetType();

// Find and call a method by name
MethodInfo? greetMethod = type.GetMethod("Greet");
greetMethod?.Invoke(person, null);   // Hello, I'm Asha

// Call a method with parameters
public class Calculator
{
    public int Add(int a, int b) => a + b;
}

var calc   = new Calculator();
var method = calc.GetType().GetMethod("Add");
object? result = method?.Invoke(calc, new object[] { 3, 4 });
Console.WriteLine(result);   // 7

// Call private method (use BindingFlags)
MethodInfo? secret = type.GetMethod("Secret",
    BindingFlags.NonPublic | BindingFlags.Instance);
secret?.Invoke(person, null);   // Shh!
```

## 🏷️ What are Attributes?

**Attributes** add **metadata** to your code — extra information the compiler, frameworks, and your own code can read.

```csharp
// You've already used attributes!
[HttpGet]           // ASP.NET Core — marks HTTP method
[Required]          // Validation — field is required
[JsonIgnore]        // JSON — skip this property
[Obsolete]          // marks method as outdated
[Fact]              // xUnit — marks a test method
```

## ✍️ Creating Custom Attributes

```csharp
// 1️⃣ Create attribute class (must end in 'Attribute')
[AttributeUsage(AttributeTargets.Class | AttributeTargets.Method)]
public class AuthorAttribute : Attribute
{
    public string Name    { get; }
    public string Version { get; }

    public AuthorAttribute(string name, string version = "1.0")
    {
        Name    = name;
        Version = version;
    }
}

// 2️⃣ Apply to your code
[Author("Asha", "2.0")]
public class OrderService
{
    [Author("Ravi")]
    public void PlaceOrder() { }
}

// 3️⃣ Read at runtime via reflection
Type type = typeof(OrderService);
var attr = type.GetCustomAttribute<AuthorAttribute>();
if (attr is not null)
    Console.WriteLine($"Author: {attr.Name}, v{attr.Version}");
// Author: Asha, v2.0
```

## 🌍 Real-World: Auto Object Printer

```csharp
// Use reflection to print ANY object's properties
public static class ObjectPrinter
{
    public static void Print(object obj)
    {
        Type type = obj.GetType();
        Console.WriteLine($"=== {type.Name} ===");

        foreach (var prop in type.GetProperties())
        {
            object? value = prop.GetValue(obj);
            Console.WriteLine($"  {prop.Name}: {value ?? "null"}");
        }
    }
}

// Works on ANY class — no changes needed ✅
var order = new Order { Id = 1, Total = 999m, CustomerEmail = "asha@test.com" };
ObjectPrinter.Print(order);
// === Order ===
//   Id: 1
//   Total: 999
//   CustomerEmail: asha@test.com

var product = new Product { Id = 5, Name = "Laptop", Price = 59999m };
ObjectPrinter.Print(product);
// === Product ===
//   Id: 5
//   Name: Laptop
//   Price: 59999
```

## 🌍 Real-World: Validation with Attributes

```csharp
// Custom validation attribute
[AttributeUsage(AttributeTargets.Property)]
public class PositiveAttribute : Attribute { }

public class Order
{
    [Positive]
    public decimal Total { get; set; }

    [Positive]
    public int Quantity { get; set; }
}

// Validator that reads attributes via reflection
public static class Validator
{
    public static List<string> Validate(object obj)
    {
        var errors = new List<string>();
        var props  = obj.GetType().GetProperties();

        foreach (var prop in props)
        {
            if (prop.GetCustomAttribute<PositiveAttribute>() is not null)
            {
                var value = Convert.ToDouble(prop.GetValue(obj));
                if (value <= 0)
                    errors.Add($"{prop.Name} must be positive");
            }
        }
        return errors;
    }
}

var order  = new Order { Total = -50, Quantity = 0 };
var errors = Validator.Validate(order);
errors.ForEach(Console.WriteLine);
// Total must be positive
// Quantity must be positive
```

## ⚠️ When to Use Reflection

```csharp
// ✅ Good uses:
// - Frameworks (ASP.NET, xUnit, EF Core use it heavily)
// - Generic object mappers (AutoMapper)
// - Serialization (System.Text.Json)
// - Plugin systems — load unknown assemblies at runtime
// - Logging/debugging tools

// ❌ Avoid for:
// - Performance-critical hot paths (reflection is slow)
// - When you know the type at compile time (use direct access)
// - Simple property access (always faster without reflection)
```

## 📋 Summary Table

| Concept | Code |
|---|---|
| Get type | `typeof(MyClass)` or `obj.GetType()` |
| Get properties | `type.GetProperties()` |
| Get methods | `type.GetMethods()` |
| Read property | `prop.GetValue(obj)` |
| Write property | `prop.SetValue(obj, value)` |
| Call method | `method.Invoke(obj, args)` |
| Get private members | `BindingFlags.NonPublic` |
| Create attribute | Inherit from `Attribute` |
| Apply attribute | `[MyAttribute]` |
| Read attribute | `type.GetCustomAttribute<T>()` |

## 🎤 Interview Q&A

**Q1: What is reflection in C#?**
A: The ability for code to inspect its own structure at runtime — read types, properties, methods, and attributes.
```csharp
Type t = typeof(Person);
Console.WriteLine(t.Name);   // Person
```

**Q2: What is `typeof` vs `GetType()`?**
A: `typeof(T)` works at compile time on a known type. `GetType()` works at runtime on an instance.
```csharp
Type t1 = typeof(Person);      // compile time
Type t2 = person.GetType();    // runtime
```

**Q3: What are attributes?**
A: Metadata added to code using `[AttributeName]` syntax. Read at runtime via reflection.
```csharp
[Required] public string Name { get; set; }
```

**Q4: How do you create a custom attribute?**
A: Inherit from `Attribute` and decorate with `[AttributeUsage]`.
```csharp
[AttributeUsage(AttributeTargets.Property)]
public class MyAttribute : Attribute { public string Info { get; set; } = ""; }
```

**Q5: How do you read an attribute at runtime?**
A: Use `GetCustomAttribute<T>()` via reflection.
```csharp
var attr = typeof(MyClass).GetCustomAttribute<MyAttribute>();
Console.WriteLine(attr?.Info);
```

**Q6: Is reflection fast?**
A: No — it's significantly slower than direct access. Use it for frameworks and tools, not performance-critical loops.

**Q7: What is `BindingFlags`?**
A: Controls which members reflection returns — e.g. `NonPublic | Instance` to include private members.
```csharp
type.GetFields(BindingFlags.NonPublic | BindingFlags.Instance);
```

**Q8: Name 3 places .NET uses reflection internally.**
A: ASP.NET Core (routing, DI), xUnit (finding `[Fact]` methods), System.Text.Json (serializing properties).

## 🚀 Key Takeaways

✅ Reflection = inspect code structure at runtime  
✅ `typeof(T)` = compile time · `obj.GetType()` = runtime  
✅ `GetProperties()`, `GetMethods()`, `GetFields()` — inspect members  
✅ `GetValue()` / `SetValue()` — read/write properties dynamically  
✅ `Invoke()` — call methods dynamically  
✅ Attributes = metadata on code, read via reflection  
✅ Create attributes by inheriting `Attribute`  
✅ Reflection is slow — use for frameworks/tools, not hot paths
