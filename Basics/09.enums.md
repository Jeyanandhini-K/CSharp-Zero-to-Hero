# 📚 Enums in C#

## 🧠 What is an Enum?

An **enum** is a set of **named constants**. It lets you define a list of related values with meaningful names instead of using raw numbers or strings.

---

## 📌 Basic Enum

```csharp
// Define enum
public enum DayOfWeek
{
    Monday,
    Tuesday,
    Wednesday,
    Thursday,
    Friday,
    Saturday,
    Sunday
}

// Use enum
DayOfWeek today = DayOfWeek.Monday;
Console.WriteLine(today);  // Monday

if (today == DayOfWeek.Monday)
{
    Console.WriteLine("Start of work week");
}
```

**Benefits:**
- ✅ Self-documenting code
- ✅ Prevents invalid values
- ✅ Type-safe

---

## 🔢 Enum Values

By default, enums start at 0 and increment.

```csharp
public enum Status
{
    Inactive,    // 0
    Active,      // 1
    Pending,     // 2
    Completed    // 3
}

Status status = Status.Active;
Console.WriteLine((int)status);  // 1
```

You can assign custom values:

```csharp
public enum Priority
{
    Low = 1,
    Medium = 5,
    High = 10,
    Critical = 100
}

Priority p = Priority.High;
Console.WriteLine((int)p);  // 10
```

## 🎯 Real-World Example: Order Status

```csharp
public enum OrderStatus
{
    Pending,
    Processing,
    Shipped,
    Delivered,
    Cancelled
}

public class Order
{
    public int OrderId { get; set; }
    public OrderStatus Status { get; set; }

    public void PrintStatus()
    {
        switch (Status)
        {
            case OrderStatus.Pending:
                Console.WriteLine("Order is pending.");
                break;
            case OrderStatus.Processing:
                Console.WriteLine("Order is being processed.");
                break;
            case OrderStatus.Shipped:
                Console.WriteLine("Order has shipped.");
                break;
            case OrderStatus.Delivered:
                Console.WriteLine("Order delivered.");
                break;
            case OrderStatus.Cancelled:
                Console.WriteLine("Order cancelled.");
                break;
        }
    }
}

Order order = new Order { OrderId = 1, Status = OrderStatus.Shipped };
order.PrintStatus();  // Order has shipped.
```

## 🔄 Convert Enum to/from String

```csharp
public enum Color { Red, Green, Blue }

// Enum to string
Color color = Color.Red;
string colorName = color.ToString();
Console.WriteLine(colorName);  // Red

// String to enum
string input = "Blue";
Color parsedColor = Enum.Parse<Color>(input);
Console.WriteLine(parsedColor);  // Blue

// Safe conversion with TryParse
if (Enum.TryParse<Color>("Green", out Color result))
{
    Console.WriteLine(result);  // Green
}
```

---

## 🔄 Convert Enum to/from Int

```csharp
public enum Priority { Low = 1, Medium = 2, High = 3 }

// Enum to int
Priority p = Priority.High;
int value = (int)p;
Console.WriteLine(value);  // 3

// Int to enum
int numValue = 2;
Priority converted = (Priority)numValue;
Console.WriteLine(converted);  // Medium
```

## 🏷️ Enum with Attributes (Flags)

Use `[Flags]` for enums that can be combined with bitwise operators.

```csharp
[Flags]
public enum Permissions
{
    Read = 1,
    Write = 2,
    Delete = 4,
    Execute = 8
}

// Combine permissions
Permissions userPerms = Permissions.Read | Permissions.Write;
Console.WriteLine(userPerms);  // Read, Write

// Check permission
if ((userPerms & Permissions.Read) == Permissions.Read)
{
    Console.WriteLine("User can read");
}

if ((userPerms & Permissions.Delete) == Permissions.Delete)
{
    Console.WriteLine("User can delete");  // Won't print
}
```

## 📋 Real-World: User Roles

```csharp
[Flags]
public enum UserRole
{
    None = 0,
    User = 1,
    Moderator = 2,
    Admin = 4
}

public class User
{
    public string Name { get; set; }
    public UserRole Roles { get; set; }

    public bool HasRole(UserRole role)
    {
        return (Roles & role) == role;
    }

    public void AddRole(UserRole role)
    {
        Roles |= role;
    }

    public void RemoveRole(UserRole role)
    {
        Roles &= ~role;
    }
}

User user = new User { Name = "Ali", Roles = UserRole.User };
user.AddRole(UserRole.Moderator);

Console.WriteLine(user.HasRole(UserRole.User));      // True
Console.WriteLine(user.HasRole(UserRole.Moderator)); // True
Console.WriteLine(user.HasRole(UserRole.Admin));     // False
```

## 🎤 Interview Q&A

### Q1: What is an enum?
**A:** Set of named constants. Makes code more readable and type-safe.

```csharp
public enum Status { Active, Inactive }

Status status = Status.Active;
```

---

### Q2: Why use enums instead of strings?
**A:** Type-safe, prevents invalid values, self-documenting.

```csharp
// ❌ Bad - strings allow typos
string status = "actve";  // Typo not caught

// ✅ Good - enum
Status status = Status.Active;  // Compiler catches errors
```

---

### Q3: What are the default enum values?
**A:** Start at 0 and increment by 1.

```csharp
public enum Color { Red, Green, Blue }
// Red = 0, Green = 1, Blue = 2
```

---

### Q4: Can you assign custom values to enum members?
**A:** Yes!

```csharp
public enum Priority
{
    Low = 1,
    Medium = 5,
    High = 10
}
```

---

### Q5: How do you convert enum to string?
**A:** Use `ToString()` or `Enum.GetName()`.

```csharp
Status status = Status.Active;
string name = status.ToString();  // "Active"
```

---

### Q6: How do you convert string to enum?
**A:** Use `Enum.Parse()` or `Enum.TryParse()`.

```csharp
Status parsed = Enum.Parse<Status>("Active");

if (Enum.TryParse<Status>("Active", out Status result))
{
    // result = Status.Active
}
```

---

### Q7: What is the [Flags] attribute?
**A:** Allows combining enum values with bitwise operators.

```csharp
[Flags]
public enum Permissions { Read = 1, Write = 2 }

Permissions perms = Permissions.Read | Permissions.Write;
```

---

### Q8: How do you check if enum has a flag?
**A:** Use bitwise AND operator `&`.

```csharp
if ((perms & Permissions.Read) == Permissions.Read)
{
    Console.WriteLine("Can read");
}
```

---

### Q9: Can you convert enum to int?
**A:** Yes! Cast it.

```csharp
Status status = Status.Active;
int value = (int)status;
```

---

### Q10: When should you use enums?
**A:** When you have a fixed set of related values (statuses, priorities, roles, states, etc.)

```csharp
// Good enum candidates
OrderStatus, UserRole, Priority, DayOfWeek, PaymentMethod
```

---

## 🚀 Key Takeaways

✅ Enum = Named constants set  
✅ Type-safe and self-documenting  
✅ Default: 0-based increment  
✅ Can assign custom values  
✅ [Flags] for combining values  
✅ Convert to/from string with Parse  
✅ Use for fixed set of values
