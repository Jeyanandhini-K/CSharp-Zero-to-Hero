# 🔀 Control Flow in C#

---

## 🧠 What is Control Flow?

Control flow decides **which code runs** and **when**. Without it, every program would just run top to bottom — no decisions, no loops, no logic.

```csharp
// Without control flow — always prints both
Console.WriteLine("You passed!");
Console.WriteLine("You failed!");

// With control flow — prints the right one
if (score >= 50)
    Console.WriteLine("You passed! ✅");
else
    Console.WriteLine("You failed! ❌");
```

## 🔀 if / else if / else

```csharp
int score = 75;

if (score >= 90)
    Console.WriteLine("Grade: A");
else if (score >= 80)
    Console.WriteLine("Grade: B");
else if (score >= 70)
    Console.WriteLine("Grade: C");
else if (score >= 60)
    Console.WriteLine("Grade: D");
else
    Console.WriteLine("Grade: F");
// Output: Grade: C
```

## 🎛️ switch Statement

```csharp
string day = "Monday";

switch (day)
{
    case "Monday":
    case "Tuesday":
    case "Wednesday":
    case "Thursday":
    case "Friday":
        Console.WriteLine("Weekday 💼");
        break;
    case "Saturday":
    case "Sunday":
        Console.WriteLine("Weekend 🎉");
        break;
    default:
        Console.WriteLine("Invalid day ❌");
        break;
}
```

## ⚡ switch Expression (Modern C#)

```csharp
// Cleaner, returns a value directly
int score = 85;
string grade = score switch
{
    >= 90 => "A",
    >= 80 => "B",
    >= 70 => "C",
    >= 60 => "D",
    _     => "F"   // _ = catch-all
};
Console.WriteLine($"Grade: {grade}");   // Grade: B
```

## 🔁 for Loop

```csharp
// for (init; condition; increment)
for (int i = 1; i <= 5; i++)
    Console.WriteLine($"Count: {i}");
// Count: 1, 2, 3, 4, 5

// Reverse
for (int i = 5; i >= 1; i--)
    Console.Write(i + " ");   // 5 4 3 2 1

// Step by 2
for (int i = 0; i <= 10; i += 2)
    Console.Write(i + " ");   // 0 2 4 6 8 10
```

## 🔁 while Loop

```csharp
// Runs while condition is true
int count = 1;
while (count <= 5)
{
    Console.WriteLine($"Count: {count}");
    count++;
}

// ✅ Use while when you don't know iterations upfront
string input = "";
while (input != "quit")
{
    Console.Write("Enter command: ");
    input = Console.ReadLine() ?? "";
}
```

---

## 🔁 do-while Loop

```csharp
// Runs AT LEAST ONCE — checks condition after
int num;
do
{
    Console.Write("Enter a positive number: ");
    num = int.Parse(Console.ReadLine() ?? "0");
} while (num <= 0);

Console.WriteLine($"You entered: {num}");
// Guaranteed to ask at least once ✅
```

## 🔁 foreach Loop

```csharp
// Best for iterating collections ✅
string[] fruits = { "Apple", "Banana", "Mango" };
foreach (string fruit in fruits)
    Console.WriteLine(fruit);

// Works on any IEnumerable
var numbers = new List<int> { 1, 2, 3, 4, 5 };
foreach (int n in numbers)
    Console.Write(n + " ");   // 1 2 3 4 5
```

## ⏭️ break, continue, return

```csharp
// break — exit loop completely
for (int i = 0; i < 10; i++)
{
    if (i == 5) break;
    Console.Write(i + " ");   // 0 1 2 3 4
}

// continue — skip this iteration
for (int i = 0; i < 10; i++)
{
    if (i % 2 == 0) continue;   // skip even
    Console.Write(i + " ");     // 1 3 5 7 9
}

// return — exit the method
int FindFirst(int[] arr, int target)
{
    foreach (int n in arr)
        if (n == target) return n;   // exit immediately
    return -1;
}
```

## 🎯 Ternary Operator

```csharp
// condition ? valueIfTrue : valueIfFalse
int age = 20;
string status = age >= 18 ? "Adult" : "Minor";
Console.WriteLine(status);   // Adult

// Nested ternary (keep it simple!)
string grade = score >= 90 ? "A" : score >= 80 ? "B" : "C";
```

## 🌍 Real-World Example: ATM Machine

```csharp
public class ATM
{
    private decimal _balance = 10000m;

    public void Withdraw(decimal amount)
    {
        if (amount <= 0)
        {
            Console.WriteLine("❌ Invalid amount");
            return;
        }

        if (amount > _balance)
        {
            Console.WriteLine("❌ Insufficient balance");
            return;
        }

        if (amount % 100 != 0)
        {
            Console.WriteLine("❌ Amount must be in multiples of 100");
            return;
        }

        _balance -= amount;
        Console.WriteLine($"✅ Withdrawn ₹{amount}. Balance: ₹{_balance}");
    }
}
```

## 📋 Summary Table

| Statement | Use when |
|---|---|
| `if/else` | Simple condition check |
| `switch` | Multiple fixed values |
| `switch expression` | Return value based on condition |
| `for` | Known number of iterations |
| `while` | Unknown iterations, check first |
| `do-while` | Must run at least once |
| `foreach` | Iterating collections |
| `break` | Exit loop early |
| `continue` | Skip current iteration |
| `return` | Exit method immediately |

## 🎤 Interview Q&A

**Q1: What is the difference between `while` and `do-while`?**
A: `while` checks condition first — may never run. `do-while` runs at least once then checks.
```csharp
while (false) { Console.WriteLine("Never"); }   // never runs
do { Console.WriteLine("Once"); } while (false); // runs once ✅
```

**Q2: When would you use `switch` over `if-else`?**
A: When comparing one variable against many fixed values. `switch` is cleaner and sometimes faster.

**Q3: What does `break` do in a loop vs switch?**
A: In both — exits the current block. In `switch` it prevents fall-through. In loops it exits the loop.

**Q4: What is the difference between `for` and `foreach`?**
A: `for` = index-based, you control iteration. `foreach` = collection-based, simpler, can't modify collection while iterating.

**Q5: What is a ternary operator?**
A: A one-line `if-else` that returns a value: `condition ? valueIfTrue : valueIfFalse`.
```csharp
string msg = age >= 18 ? "Adult" : "Minor";
```

## 🚀 Key Takeaways

✅ `if/else` — basic decision making  
✅ `switch expression` — modern, clean, returns a value  
✅ `for` — when you know the count  
✅ `foreach` — best for collections  
✅ `while` — when count is unknown  
✅ `do-while` — guaranteed at least one run  
✅ `break` exits loop · `continue` skips iteration · `return` exits method
