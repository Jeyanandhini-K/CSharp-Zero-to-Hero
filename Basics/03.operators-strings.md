# Operators and String Manipulation

Operators perform operations on variables. Strings are sequences of characters (text).

ARITHMETIC OPERATORS

+ Addition: 10 + 5 = 15
- Subtraction: 10 - 5 = 5
* Multiplication: 10 * 5 = 50
/ Division: 10 / 5 = 2
% Modulus (remainder): 10 % 3 = 1
++ Increment: x++ or ++x increases by 1
-- Decrement: x-- or --x decreases by 1

Examples:

int a = 10;
int b = 3;

int sum = a + b;           // 13
int difference = a - b;    // 7
int product = a * b;       // 30
int quotient = a / b;      // 3
int remainder = a % b;     // 1

Console.WriteLine($"Sum: {sum}, Remainder: {remainder}");
// Output: Sum: 13, Remainder: 1

Increment and Decrement:

int x = 5;

x++;  // x becomes 6
x--;  // x becomes 5

++x;  // Pre-increment (increment, then use)
x++;  // Post-increment (use, then increment)

Console.WriteLine(x);  // 6

ASSIGNMENT OPERATORS

= Assign: x = 5
+= Add and assign: x += 3 means x = x + 3
-= Subtract and assign: x -= 2 means x = x - 2
*= Multiply and assign: x *= 4 means x = x * 4
/= Divide and assign: x /= 2 means x = x / 2
%= Modulus and assign: x %= 3 means x = x % 3

Examples:

int x = 10;

x += 5;   // x = 15
x -= 3;   // x = 12
x *= 2;   // x = 24
x /= 4;   // x = 6

Console.WriteLine(x);  // 6

COMPARISON OPERATORS

Review from Control Flow:
== Equal: 5 == 5 is true
!= Not equal: 5 != 3 is true
> Greater than: 5 > 3 is true
< Less than: 5 < 3 is false
>= Greater or equal: 5 >= 5 is true
<= Less or equal: 5 <= 3 is false

LOGICAL OPERATORS

Review from Control Flow:
&& AND (both true): true && false is false
|| OR (at least one true): true || false is true
! NOT (reverses): !true is false

Examples:

int age = 25;
bool hasLicense = true;
bool hasInsurance = true;

if (age >= 18 && hasLicense && hasInsurance) {
    Console.WriteLine("You can drive!");
}
// Output: You can drive!

bool canVote = age >= 18 || age == 17;
Console.WriteLine(canVote);  // true

bool isValidAge = !(age < 0 || age > 120);
Console.WriteLine(isValidAge);  // true

STRING BASICS

A string is a sequence of characters enclosed in double quotes.

String Declaration:

string name = "Ahmed Khan";
string message = "Hello, C#!";
string empty = "";
string? nullable = null;  // Can be null with ?

STRING CONCATENATION - COMBINING STRINGS

Method 1: Using + (older way)

string firstName = "Ali";
string lastName = "Khan";

string fullName = firstName + " " + lastName;
Console.WriteLine(fullName);  // Ali Khan

Method 2: String Interpolation - BEST WAY

string greeting = $"Hello, {firstName} {lastName}!";
Console.WriteLine(greeting);  // Hello, Ali Khan!

Method 3: String.Format

string message = string.Format("Welcome, {0}!", firstName);
Console.WriteLine(message);  // Welcome, Ali!

PREFERRED: Use string interpolation with $ because it's cleaner and more modern.

STRING PROPERTIES AND METHODS

Length - Get number of characters:

string text = "C# Programming";
int length = text.Length;
Console.WriteLine(length);  // 14 (includes space)

ToUpper - Convert to uppercase:

string text = "Hello World";
Console.WriteLine(text.ToUpper());  // HELLO WORLD

ToLower - Convert to lowercase:

Console.WriteLine(text.ToLower());  // hello world

Substring - Extract part of string:

string text = "Pakistan";
string part = text.Substring(0, 3);  // Start at index 0, take 3 chars
Console.WriteLine(part);  // Pak

string part2 = text.Substring(4);  // From index 4 to end
Console.WriteLine(part2);  // stan

IndexOf - Find position of substring:

string text = "Hello World";
int position = text.IndexOf("World");
Console.WriteLine(position);  // 6

int notFound = text.IndexOf("xyz");
Console.WriteLine(notFound);  // -1 (not found)

Contains - Check if substring exists:

string email = "user@example.com";

if (email.Contains("@")) {
    Console.WriteLine("Valid email format");
}
// Output: Valid email format

Replace - Replace text:

string text = "I like Java, Java is cool";
string newText = text.Replace("Java", "C#");
Console.WriteLine(newText);
// Output: I like C#, C# is cool

Split - Break into parts:

string csv = "Ali,Ahmed,Hassan";
string[] names = csv.Split(',');

foreach (string name in names) {
    Console.WriteLine(name);
}
// Output:
// Ali
// Ahmed
// Hassan

Trim - Remove spaces:

string text = "  Hello World  ";
Console.WriteLine(text.Trim());     // "Hello World"
Console.WriteLine(text.TrimStart()); // "Hello World  "
Console.WriteLine(text.TrimEnd());   // "  Hello World"

StartsWith and EndsWith:

string url = "https://www.example.com";

if (url.StartsWith("https")) {
    Console.WriteLine("Secure connection");
}

if (url.EndsWith(".com")) {
    Console.WriteLine("Commercial domain");
}

STRING COMPARISON

Exact Match:

string password1 = "MyPassword123";
string password2 = "MyPassword123";

if (password1 == password2) {
    Console.WriteLine("Passwords match");
}
// Output: Passwords match

Case-Insensitive Comparison - Method 1:

string country1 = "Pakistan";
string country2 = "PAKISTAN";

if (country1.ToLower() == country2.ToLower()) {
    Console.WriteLine("Same country");
}

Case-Insensitive Comparison - Method 2:

if (country1.Equals(country2, StringComparison.OrdinalIgnoreCase)) {
    Console.WriteLine("Same country");
}

TERNARY OPERATOR - SHORTHAND IF/ELSE

Assign value based on condition in one line.

Syntax:
variable = condition ? valueIfTrue : valueIfFalse;

Examples:

int age = 20;
string status = age >= 18 ? "Adult" : "Minor";
Console.WriteLine(status);  // Adult

int score = 75;
string result = score >= 50 ? "Pass" : "Fail";
Console.WriteLine(result);  // Pass

Nested Ternary:

int num = 10;
string category = num > 10 ? "Above 10" : num == 10 ? "Exactly 10" : "Below 10";
Console.WriteLine(category);  // Exactly 10

REAL-WORLD EXAMPLE - USER REGISTRATION

string username = "ali_khan";
string password = "SecurePass123!";
int age = 22;

// Validate username
bool hasValidUsername = username.Length >= 5 && username.Contains("_");
string usernameStatus = hasValidUsername ? "Valid" : "Invalid";

// Validate password strength
bool hasStrongPassword = password.Length >= 8 && (password.Contains("!") || password.Contains("@"));
string passwordStatus = hasStrongPassword ? "Strong" : "Weak";

// Validate age
bool isAdult = age >= 18;
string ageStatus = isAdult ? "Adult" : "Minor";

// Overall registration
if (hasValidUsername && hasStrongPassword && isAdult) {
    Console.WriteLine("Registration successful!");
    Console.WriteLine($"Welcome, {username.ToUpper()}!");
}
else {
    Console.WriteLine("Registration failed");
    Console.WriteLine($"Username: {usernameStatus}");
    Console.WriteLine($"Password: {passwordStatus}");
    Console.WriteLine($"Age: {ageStatus}");
}

Output:
Registration successful!
Welcome, ALI_KHAN!

INTERVIEW QUESTIONS

Q: What's difference between / and % operators?
A: / divides and returns quotient. % returns remainder after division.

Q: When use string interpolation vs concatenation?
A: String interpolation ($"...") is cleaner and more modern. Use it always.

Q: What does Substring(0, 3) do?
A: Returns first 3 characters starting at index 0.

Q: How check if string contains a word?
A: Use .Contains("word") method.

Q: When use ternary operator?
A: For simple if/else assignments in one line. Keeps code concise.

Q: What's difference between == and .Equals() for strings?
A: Both check value equality for strings. .Equals() is safer for null checks.

Q: Why use decimal for money but not double for string operations?
A: Strings are immutable and work fine. But for calculations, decimal prevents rounding errors.

KEY TAKEAWAYS

- Arithmetic operators: +, -, *, /, %
- Assignment operators: +=, -=, *=, /=
- String interpolation: $"Hello, {name}!"
- String methods: .Length, .ToUpper(), .Contains(), .Replace(), .Split(), .Trim()
- Ternary operator: condition ? yes : no
- Always validate string input before using
- String concatenation creates new strings (use StringBuilder for many concatenations)
- Use meaningful operator combinations for complex logic
