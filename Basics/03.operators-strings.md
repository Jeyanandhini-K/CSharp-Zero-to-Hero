# ➕ Operators & Strings in C#

---

## 🧠 What are Operators?

Operators perform operations on values. C# has operators for math, comparison, logic, and more.

---

## ➕ Arithmetic Operators

```csharp
int a = 10, b = 3;

Console.WriteLine(a + b);   // 13 — addition
Console.WriteLine(a - b);   // 7  — subtraction
Console.WriteLine(a * b);   // 30 — multiplication
Console.WriteLine(a / b);   // 3  — division (integer!)
Console.WriteLine(a % b);   // 1  — modulus (remainder)

// ⚠️ Integer division truncates!
Console.WriteLine(10 / 3);    // 3 — not 3.33!
Console.WriteLine(10.0 / 3);  // 3.333... — use double ✅

// Power
double squared = Math.Pow(2, 10);   // 1024
double root    = Math.Sqrt(16);     // 4
```

---

## 📈 Assignment Operators

```csharp
int x = 10;

x += 5;    // x = x + 5  → 15
x -= 3;    // x = x - 3  → 12
x *= 2;    // x = x * 2  → 24
x /= 4;    // x = x / 4  → 6
x %= 4;    // x = x % 4  → 2

// Increment / Decrement
int count = 5;
count++;   // 6 — post-increment
++count;   // 7 — pre-increment
count--;   // 6 — post-decrement
--count;   // 5 — pre-decrement

// Pre vs Post difference
int a = 5;
Console.WriteLine(a++);   // 5 — prints THEN increments
Console.WriteLine(a);     // 6

int b = 5;
Console.WriteLine(++b);   // 6 — increments THEN prints
```

## 🔍 Comparison Operators

```csharp
int x = 10, y = 20;

Console.WriteLine(x == y);   // False — equal
Console.WriteLine(x != y);   // True  — not equal
Console.WriteLine(x >  y);   // False — greater than
Console.WriteLine(x <  y);   // True  — less than
Console.WriteLine(x >= y);   // False — greater or equal
Console.WriteLine(x <= y);   // True  — less or equal

// All return bool ✅
bool isAdult = age >= 18;
bool isEmpty = name == "";
```

## 🔗 Logical Operators

```csharp
bool a = true, b = false;

Console.WriteLine(a && b);   // False — AND (both must be true)
Console.WriteLine(a || b);   // True  — OR  (at least one true)
Console.WriteLine(!a);       // False — NOT (reverses)

// Real example
int age = 25;
bool hasLicense = true;

bool canDrive = age >= 18 && hasLicense;   // True ✅

// Short-circuit evaluation
// && stops if first is false
// || stops if first is true
string? name = null;
if (name != null && name.Length > 0)   // safe — won't crash ✅
    Console.WriteLine(name);
```

## 🔢 Bitwise Operators

```csharp
int a = 0b1010;   // 10 in binary
int b = 0b1100;   // 12 in binary

Console.WriteLine(a & b);    // 0b1000 = 8  — AND
Console.WriteLine(a | b);    // 0b1110 = 14 — OR
Console.WriteLine(a ^ b);    // 0b0110 = 6  — XOR
Console.WriteLine(~a);       // bitwise NOT
Console.WriteLine(a << 1);   // 0b10100 = 20 — left shift (×2)
Console.WriteLine(a >> 1);   // 0b0101  = 5  — right shift (÷2)
```

## 🔤 String Operations

```csharp
string name = "Asha";

// Concatenation
string greeting = "Hello, " + name + "!";   // Hello, Asha!

// Interpolation ✅ (preferred)
string msg = $"Hello, {name}! You are {age} years old.";

// Verbatim string — no escape needed
string path = @"C:\Users\Asha\Documents";   // ✅
// vs
string path2 = "C:\\Users\\Asha\\Documents"; // ❌ messy

// Raw string literal (C# 11)
string json = """
{
  "name": "Asha",
  "age": 30
}
""";
```

## 🛠️ String Methods

```csharp
string s = "  Hello, World!  ";

// Case
Console.WriteLine(s.ToUpper());          // HELLO, WORLD!
Console.WriteLine(s.ToLower());          // hello, world!

// Trim
Console.WriteLine(s.Trim());             // Hello, World!
Console.WriteLine(s.TrimStart());        // Hello, World!  (left only)
Console.WriteLine(s.TrimEnd());          //   Hello, World! (right only)

// Search
Console.WriteLine(s.Contains("World")); // True
Console.WriteLine(s.StartsWith("  H")); // True
Console.WriteLine(s.EndsWith("!  "));   // True
Console.WriteLine(s.IndexOf("World"));  // 9

// Modify
Console.WriteLine(s.Replace("World", "C#"));  // Hello, C#!
Console.WriteLine(s.Trim().Substring(0, 5));   // Hello

// Split & Join
string csv = "Asha,Ravi,Priya";
string[] names = csv.Split(',');              // ["Asha","Ravi","Priya"]
string joined  = string.Join(" | ", names);  // Asha | Ravi | Priya

// Check empty
Console.WriteLine(string.IsNullOrEmpty(""));       // True
Console.WriteLine(string.IsNullOrWhiteSpace("  ")); // True
```

## 🎯 String Formatting

```csharp
decimal price = 1234567.89m;
double  pi    = 3.14159265;
int     count = 42;

Console.WriteLine($"{price:C}");     // ₹12,34,567.89 (currency)
Console.WriteLine($"{price:N2}");    // 1,234,567.89  (number, 2 decimals)
Console.WriteLine($"{pi:F3}");       // 3.142         (fixed, 3 decimals)
Console.WriteLine($"{count:D5}");    // 00042          (padded digits)
Console.WriteLine($"{0.75:P}");      // 75.00%         (percentage)
```

## 🌍 Real-World Example

```csharp
public class OrderSummary
{
    public string   CustomerName { get; set; } = "";
    public decimal  Total        { get; set; }
    public int      Items        { get; set; }
    public bool     IsPremium    { get; set; }

    public string GetSummary()
    {
        decimal discount = IsPremium ? Total * 0.10m : 0;
        decimal final    = Total - discount;

        return $"""
            Customer : {CustomerName.ToUpper()}
            Items    : {Items}
            Total    : {Total:C}
            Discount : {discount:C} {(IsPremium ? "(Premium 10%)" : "")}
            Final    : {final:C}
            """;
    }
}
```

## 🎤 Interview Q&A

**Q1: What is the difference between `==` and `.Equals()` for strings?**
A: For strings both compare values. `==` is overridden in string to compare content, not reference.
```csharp
string a = "hello"; string b = "hello";
Console.WriteLine(a == b);       // True ✅
Console.WriteLine(a.Equals(b));  // True ✅
```

**Q2: What is the modulus operator and when is it useful?**
A: `%` returns the remainder. Used to check even/odd, divisibility, and cycling through values.
```csharp
Console.WriteLine(10 % 3);   // 1
bool isEven = n % 2 == 0;    // even/odd check
int slot = index % arraySize; // cycling ✅
```

**Q3: What is string interpolation vs concatenation?**
A: Interpolation (`$"..."`) is cleaner and more readable. Concatenation (`+`) creates intermediate strings.
```csharp
string name = "Asha";
string a = "Hello " + name + "!";     // concatenation
string b = $"Hello {name}!";          // interpolation ✅
```

**Q4: What is a verbatim string?**
A: Prefixed with `@` — no escape sequences needed. Great for file paths and regex.
```csharp
string path = @"C:\Users\Asha";   // ✅
string bad  = "C:\\Users\\Asha";  // ❌ messy
```

**Q5: What is short-circuit evaluation?**
A: `&&` stops if the first condition is false. `||` stops if the first is true. Prevents unnecessary evaluation.
```csharp
string? s = null;
if (s != null && s.Length > 0) { }   // safe — won't throw ✅
```

## 🚀 Key Takeaways

✅ `%` modulus returns remainder — great for even/odd checks  
✅ Integer division truncates — use `double` when you need decimals  
✅ String interpolation `$"..."` is cleaner than concatenation `+`  
✅ `@"..."` verbatim strings — no escaping needed  
✅ `&&` and `||` short-circuit — put cheap checks first  
✅ `string.IsNullOrEmpty` and `IsNullOrWhiteSpace` for null safety  
✅ Format strings: `:C` currency · `:N2` number · `:F2` fixed · `:P` percent
