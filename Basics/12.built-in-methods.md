## 🔤 STRING METHODS

---

### ✂️ Trimming & Cleaning

```csharp
string s = "  Hello, World!  ";

s.Trim();         // "Hello, World!"    — remove both sides
s.TrimStart();    // "Hello, World!  "  — remove left only
s.TrimEnd();      // "  Hello, World!"  — remove right only

// Remove specific characters
"***hello***".Trim('*');   // "hello"
"000123".TrimStart('0');   // "123"
```

---

### 🔠 Case Conversion

```csharp
"hello".ToUpper();         // "HELLO"
"HELLO".ToLower();         // "hello"

// Culture-invariant (use for comparisons)
"hello".ToUpperInvariant(); // "HELLO"
"HELLO".ToLowerInvariant(); // "hello"
```

---

### 🔍 Searching

```csharp
string s = "Hello, World!";

s.Contains("World");       // True
s.Contains("world");       // False — case sensitive!
s.Contains("world", StringComparison.OrdinalIgnoreCase); // True ✅

s.StartsWith("Hello");     // True
s.EndsWith("!");           // True

s.IndexOf("World");        // 7  — first position
s.LastIndexOf("l");        // 10 — last position
s.IndexOf("xyz");          // -1 — not found
```

---

### ✂️ Slicing & Extracting

```csharp
string s = "Hello, World!";

s.Substring(7);        // "World!" — from index 7
s.Substring(7, 5);     // "World"  — 5 chars from index 7
s[7..12];              // "World"  — range syntax (C# 8+) ✅
s[..5];                // "Hello"  — first 5
s[^6..];               // "World!" — last 6

// Remove
s.Remove(5);           // "Hello"     — remove from index 5
s.Remove(5, 2);        // "HelloWorld!" — remove 2 chars from index 5
```

### 🔄 Replacing

```csharp
"Hello World".Replace("World", "C#");   // "Hello C#"
"aabbcc".Replace("b", "");              // "aacc" — delete
"Hello World".Replace('o', '0');        // "Hell0 W0rld"

// Replace with comparison
string result = "Hello World"
    .Replace("WORLD", "C#", StringComparison.OrdinalIgnoreCase);
// "Hello C#" ✅
```

### ✂️ Split & Join

```csharp
// Split
string csv = "Asha,Ravi,Priya,Kiran";
string[] names = csv.Split(',');
// ["Asha", "Ravi", "Priya", "Kiran"]

// Split with multiple separators
"one two  three".Split(' ', StringSplitOptions.RemoveEmptyEntries);
// ["one", "two", "three"]

// Split by string
"a--b--c".Split("--");   // ["a", "b", "c"]

// Join
string joined = string.Join(", ", names);   // "Asha, Ravi, Priya, Kiran"
string path   = string.Join("/", "usr", "local", "bin"); // "usr/local/bin"
```

### 🔢 Padding & Formatting

```csharp
"42".PadLeft(5);          // "   42"  — right-align
"42".PadRight(5);         // "42   "  — left-align
"42".PadLeft(5, '0');     // "00042"  — zero-pad ✅
"Hi".PadRight(10, '-');   // "Hi--------"

// String formatting
string.Format("Hello, {0}! You are {1} years old.", "Asha", 30);
// "Hello, Asha! You are 30 years old."

// Interpolation (preferred ✅)
$"Hello, {"Asha"}! Price: {999.99m:C}";
```

## 🔢 CONVERT METHODS

```csharp
// To numbers
Convert.ToInt32("42");        // 42
Convert.ToDouble("3.14");     // 3.14
Convert.ToDecimal("9.99");    // 9.99m
Convert.ToInt64("99999999");  // long

// Convert.ToInt32(null) = 0  ✅ (doesn't throw)
// int.Parse(null)       = ❌ throws!

// To string
Convert.ToString(42);         // "42"
Convert.ToString(true);       // "True"
Convert.ToString(3.14);       // "3.14"

// To bool
Convert.ToBoolean(1);         // True
Convert.ToBoolean(0);         // False
Convert.ToBoolean("true");    // True

// To char
Convert.ToChar(65);           // 'A'
Convert.ToChar('A');          // 'A'

// Safe parse — no exception on failure ✅
bool ok = int.TryParse("abc", out int result);
Console.WriteLine(ok);        // False
Console.WriteLine(result);    // 0 — default

double.TryParse("3.14", out double d);
bool.TryParse("true", out bool b);
DateTime.TryParse("2024-01-15", out DateTime dt);
```

## 🌍 Real-World: String Processing Utility

```csharp
public static class StringHelper
{
    // Title case: "hello world" → "Hello World"
    public static string ToTitleCase(string input)
        => string.Join(" ",
            input.Split(' ')
                 .Select(w => w.Length > 0
                     ? char.ToUpper(w[0]) + w[1..].ToLower()
                     : w));

    // Truncate with ellipsis
    public static string Truncate(string input, int maxLength)
        => input.Length <= maxLength
            ? input
            : input[..maxLength] + "...";

    // Count words
    public static int WordCount(string input)
        => input.Split(' ', StringSplitOptions.RemoveEmptyEntries).Length;

    // Is valid number
    public static bool IsNumeric(string input)
        => double.TryParse(input, out _);

    // Remove all whitespace
    public static string RemoveSpaces(string input)
        => new string(input.Where(c => !char.IsWhiteSpace(c)).ToArray());

    // Repeat string
    public static string Repeat(string input, int times)
        => string.Concat(Enumerable.Repeat(input, times));
}

// Usage
Console.WriteLine(StringHelper.ToTitleCase("hello world c#")); // Hello World C#
Console.WriteLine(StringHelper.Truncate("Long text here", 8)); // Long tex...
Console.WriteLine(StringHelper.WordCount("hello world"));       // 2
Console.WriteLine(StringHelper.IsNumeric("3.14"));             // True
Console.WriteLine(StringHelper.Repeat("Ha", 3));               // HaHaHa
```

## 🎤 Interview Q&A

**Q1: What is the difference between `String.IsNullOrEmpty` and `String.IsNullOrWhiteSpace`?**
A: `IsNullOrEmpty` catches null and `""`. `IsNullOrWhiteSpace` also catches `"   "` (spaces only).
```csharp
string.IsNullOrEmpty("  ");      // False — has spaces
string.IsNullOrWhiteSpace("  "); // True  — only whitespace ✅
```

**Q2: What is the difference between `Math.Round(2.5)` and what you'd expect?**
A: Banker's rounding — rounds to the nearest even number. `Math.Round(2.5)` = 2, not 3!
```csharp
Math.Round(2.5);   // 2 — rounds to even ⚠️
Math.Round(3.5);   // 4 — rounds to even
Math.Round(2.5, MidpointRounding.AwayFromZero);  // 3 ✅
```

**Q3: What is the difference between `Convert.ToInt32` and `int.Parse`?**
A: `Convert.ToInt32(null)` = 0. `int.Parse(null)` throws. `int.TryParse` is safest — no exception ever.
```csharp
Convert.ToInt32(null);           // 0 — safe
int.Parse(null);                 // ❌ throws
int.TryParse("abc", out int n);  // false, n=0 — safest ✅
```

**Q4: How do you sort a list of strings alphabetically?**
```csharp
var names = new List<string> { "Priya", "Asha", "Ravi" };
names.Sort();   // ["Asha", "Priya", "Ravi"] ✅
```

**Q5: What is `Math.Clamp`?**
A: Restricts a value between a min and max — great for validating ranges.
```csharp
Math.Clamp(150, 0, 100);   // 100 — clamped to max
Math.Clamp(-5,  0, 100);   // 0   — clamped to min
Math.Clamp(50,  0, 100);   // 50  — within range ✅
```

**Q6: How do you get a random number in a range?**
```csharp
int dice = Random.Shared.Next(1, 7);   // 1 to 6 ✅
```

**Q7: What does `string.Split` do with `RemoveEmptyEntries`?**
A: Removes empty strings from the result when splitting on multiple consecutive separators.
```csharp
"a,,b,,c".Split(',');  // ["a","","b","","c"] — empty strings included
"a,,b,,c".Split(',', StringSplitOptions.RemoveEmptyEntries);  // ["a","b","c"] ✅
```

**Q8: What is `PadLeft` used for?**
A: Right-aligns text by padding with spaces or a character — great for formatting tables.
```csharp
"42".PadLeft(5);       // "   42"
"42".PadLeft(5, '0');  // "00042" ✅ — zero padding
```

## 🚀 Key Takeaways

✅ `Trim` / `TrimStart` / `TrimEnd` — remove whitespace  
✅ `Contains` / `StartsWith` / `EndsWith` — search strings  
✅ `Replace` / `Remove` / `Insert` — modify strings  
✅ `Split` / `Join` — convert between string and array  
✅ `IsNullOrWhiteSpace` — safest empty check  
✅ `Math.Clamp` — restrict value to a range  
✅ `Math.Round(..., AwayFromZero)` — avoid banker's rounding surprise  
✅ `int.TryParse` — always safer than `int.Parse`  
✅ `Random.Shared.Next()` — no need to create a Random instance  
✅ `StringBuilder` — for building large strings in loops
