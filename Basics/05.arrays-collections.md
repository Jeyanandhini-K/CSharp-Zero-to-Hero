# Arrays and Collections

Arrays and Collections store multiple values. Essential for handling data.

ARRAYS - FIXED SIZE

An array is a collection of same-type elements stored in order. Size is fixed when created.

Declare and Initialize:

int[] scores = { 85, 90, 78, 92 };
string[] countries = { "Pakistan", "India", "USA" };
double[] prices = new double[3];  // Empty array of size 3

Access Elements (0-indexed):

Console.WriteLine(scores[0]);    // 85 (first)
Console.WriteLine(scores[2]);    // 78 (third)
Console.WriteLine(scores[3]);    // 92 (last)

scores[0] = 95;  // Change first element
Console.WriteLine(scores[0]);    // 95

Array Length:

Console.WriteLine(scores.Length);  // 4

Loop Through Array - For Loop:

for (int i = 0; i < scores.Length; i++) {
    Console.WriteLine(scores[i]);
}
// Output: 85, 90, 78, 92

Loop Through Array - Foreach Loop:

foreach (int score in scores) {
    Console.WriteLine(score);
}

Find Average:

int[] scores = { 85, 90, 78, 92, 88 };
int sum = 0;

foreach (int score in scores) {
    sum += score;
}

double average = sum / (double)scores.Length;
Console.WriteLine($"Average: {average}");  // Average: 86.6

TWO-DIMENSIONAL ARRAYS - MATRIX

Arrays with rows and columns.

Declare 2D Array:

int[,] matrix = new int[3, 3];  // 3x3 matrix

Initialize with Values:

int[,] grid = {
    { 1, 2, 3 },
    { 4, 5, 6 },
    { 7, 8, 9 }
};

Access Elements:

Console.WriteLine(grid[0, 0]);  // 1 (row 0, col 0)
Console.WriteLine(grid[1, 2]);  // 6 (row 1, col 2)
Console.WriteLine(grid[2, 1]);  // 8 (row 2, col 1)

Loop Through 2D Array:

for (int row = 0; row < matrix.GetLength(0); row++) {
    for (int col = 0; col < matrix.GetLength(1); col++) {
        Console.WriteLine(matrix[row, col]);
    }
}

JAGGED ARRAYS - ARRAY OF ARRAYS

Arrays with varying lengths.

Declare Jagged Array:

int[][] jagged = new int[3][];

jagged[0] = new int[2];  // Row 0 has 2 elements
jagged[1] = new int[4];  // Row 1 has 4 elements
jagged[2] = new int[3];  // Row 2 has 3 elements

Initialize Jagged Array:

int[][] data = {
    new int[] { 1, 2 },
    new int[] { 3, 4, 5 },
    new int[] { 6 }
};

Console.WriteLine(data[0][0]);  // 1
Console.WriteLine(data[1][2]);  // 5

LIST - DYNAMIC COLLECTION

A List is like a dynamic array - it grows and shrinks as needed.

Declare and Add Elements:

using System.Collections.Generic;

List<int> numbers = new List<int>();
numbers.Add(10);
numbers.Add(20);
numbers.Add(30);

Console.WriteLine(numbers.Count);  // 3

foreach (int num in numbers) {
    Console.WriteLine(num);
}

Initialize with Values:

List<string> fruits = new List<string> { "Apple", "Banana", "Orange" };

Console.WriteLine(fruits[0]);  // Apple
Console.WriteLine(fruits.Count);  // 3

Common List Methods:

List<int> nums = new List<int> { 5, 10, 15, 20 };

nums.Add(25);              // Add element
nums.Insert(0, 3);         // Insert at beginning
nums.Remove(15);           // Remove element
nums.RemoveAt(0);          // Remove at index 0

if (nums.Contains(10)) {
    Console.WriteLine("10 is in list");
}

int index = nums.IndexOf(20);  // Get index of 20

Sort and Reverse:

List<int> nums = new List<int> { 5, 2, 8, 1, 9 };

nums.Sort();      // Ascending: 1, 2, 5, 8, 9
foreach (int n in nums) Console.Write(n + " ");

nums.Reverse();   // Descending: 9, 8, 5, 2, 1
foreach (int n in nums) Console.Write(n + " ");

DICTIONARY - KEY-VALUE PAIRS

Store data as key-value pairs. Fast lookups by key.

Declare and Add:

Dictionary<string, int> ages = new Dictionary<string, int>();

ages.Add("Ali", 25);
ages.Add("Fatima", 23);
ages.Add("Hassan", 28);

Console.WriteLine(ages["Ali"]);  // 25

Initialize with Values:

Dictionary<string, string> capitals = new Dictionary<string, string> {
    { "Pakistan", "Islamabad" },
    { "India", "New Delhi" },
    { "USA", "Washington DC" }
};

Console.WriteLine(capitals["Pakistan"]);  // Islamabad

Common Dictionary Methods:

Dictionary<string, int> scores = new Dictionary<string, int> {
    { "Ali", 85 },
    { "Ahmed", 90 }
};

Check if Key Exists:

if (scores.ContainsKey("Ali")) {
    Console.WriteLine("Ali's score: " + scores["Ali"]);
}

Loop Through:

foreach (var entry in scores) {
    Console.WriteLine($"{entry.Key}: {entry.Value}");
}

Get Value Safely:

if (scores.TryGetValue("Fatima", out int score)) {
    Console.WriteLine(score);
} else {
    Console.WriteLine("Student not found");
}

Remove:

scores.Remove("Ali");

Count:

Console.WriteLine(scores.Count);

QUEUE - FIFO (First In, First Out)

First item added is first item removed.

Basic Example:

Queue<int> queue = new Queue<int>();

queue.Enqueue(10);  // Add to back
queue.Enqueue(20);
queue.Enqueue(30);

int first = queue.Dequeue();  // Remove from front: 10
Console.WriteLine(first);

int peek = queue.Peek();   // Look at front without removing: 20

Real-World Use Case - Print Queue:

Queue<string> printQueue = new Queue<string>();

printQueue.Enqueue("Document1.pdf");
printQueue.Enqueue("Document2.pdf");
printQueue.Enqueue("Document3.pdf");

while (printQueue.Count > 0) {
    string doc = printQueue.Dequeue();
    Console.WriteLine($"Printing: {doc}");
}

STACK - LIFO (Last In, First Out)

Last item added is first item removed.

Basic Example:

Stack<int> stack = new Stack<int>();

stack.Push(10);   // Add to top
stack.Push(20);
stack.Push(30);

int top = stack.Pop();    // Remove from top: 30
Console.WriteLine(top);

int peek = stack.Peek();  // Look at top: 20

Real-World Use Case - Undo/Redo:

Stack<string> undoStack = new Stack<string>();

undoStack.Push("Typed: Hello");
undoStack.Push("Typed: World");
undoStack.Push("Deleted: d");

while (undoStack.Count > 0) {
    string action = undoStack.Pop();
    Console.WriteLine($"Undo: {action}");
}

PRACTICAL EXAMPLES

Find Highest Score:

int[] scores = { 85, 92, 78, 95, 88 };
int highest = scores[0];

foreach (int score in scores) {
    if (score > highest) {
        highest = score;
    }
}

Console.WriteLine($"Highest: {highest}");  // 95

Student Management:

Dictionary<string, double> students = new Dictionary<string, double> {
    { "Ali", 3.8 },
    { "Fatima", 3.5 },
    { "Hassan", 3.9 }
};

double bestGPA = 0;
string topStudent = "";

foreach (var student in students) {
    if (student.Value > bestGPA) {
        bestGPA = student.Value;
        topStudent = student.Key;
    }
}

Console.WriteLine($"Top Student: {topStudent} ({bestGPA})");
// Output: Top Student: Hassan (3.9)

Remove Duplicates from List:

List<int> numbers = new List<int> { 5, 2, 5, 8, 2, 9, 5 };

List<int> unique = new List<int>();

foreach (int num in numbers) {
    if (!unique.Contains(num)) {
        unique.Add(num);
    }
}

Console.WriteLine($"Unique: {string.Join(", ", unique)}");
// Output: Unique: 5, 2, 8, 9

ARRAY vs LIST

Array:
- Fixed size
- Faster
- Cannot add/remove after creation
- Use when size is known

List:
- Dynamic size
- Slightly slower
- Can add/remove items
- Use when size changes

INTERVIEW QUESTIONS

Q: Difference between array and list?
A: Array has fixed size. List is dynamic (grows/shrinks).

Q: How access 3rd element of array?
A: array[2] (0-indexed: 0, 1, 2...)

Q: When use Dictionary instead of List?
A: When you need key-value lookups (like student ID -> name).

Q: Difference between Queue and Stack?
A: Queue is FIFO (first in, first out). Stack is LIFO (last in, first out).

Q: How get length of List?
A: .Count property

Q: Real-world use of Queue?
A: Print queue, customer service queue, task scheduling.

Q: Real-world use of Stack?
A: Undo/Redo functionality, function call stack, browser back button.

KEY TAKEAWAYS

- Array = fixed size, stores same type
- List = dynamic size, stores same type
- Dictionary = key-value pairs for fast lookups
- Queue = FIFO (first in, first out)
- Stack = LIFO (last in, first out)
- Use .Length for arrays
- Use .Count for lists
- foreach loops work great with collections
- 2D arrays = matrix (rows and columns)
- Always check bounds before accessing array elements
