# 📚 Arrays & Collections in C#

---

## 🧠 What are Arrays?

An **array** stores multiple values of the **same type** in a fixed-size container.

```csharp
// Declare and initialize
int[]    numbers = { 1, 2, 3, 4, 5 };
string[] names   = { "Asha", "Ravi", "Priya" };
double[] prices  = new double[3];   // empty, size 3

// Access by index (starts at 0!)
Console.WriteLine(numbers[0]);   // 1 — first
Console.WriteLine(numbers[4]);   // 5 — last
Console.WriteLine(numbers[^1]);  // 5 — last (C# 8+ syntax ✅)

// Modify
numbers[0] = 99;
Console.WriteLine(numbers[0]);   // 99
```

## 🔢 Array Operations

```csharp
int[] nums = { 5, 3, 8, 1, 9, 2 };

// Length
Console.WriteLine(nums.Length);   // 6

// Sort
Array.Sort(nums);
Console.WriteLine(string.Join(", ", nums));   // 1, 2, 3, 5, 8, 9

// Reverse
Array.Reverse(nums);
Console.WriteLine(string.Join(", ", nums));   // 9, 8, 5, 3, 2, 1

// Search
int idx = Array.IndexOf(nums, 5);   // position of 5
bool has = Array.Exists(nums, n => n > 7);   // True

// Copy
int[] copy = new int[nums.Length];
Array.Copy(nums, copy, nums.Length);

// Fill
int[] filled = new int[5];
Array.Fill(filled, 42);   // { 42, 42, 42, 42, 42 }
```

## 📐 2D Arrays

```csharp
// Matrix — rows × columns
int[,] matrix = {
    { 1, 2, 3 },
    { 4, 5, 6 },
    { 7, 8, 9 }
};

Console.WriteLine(matrix[0, 0]);   // 1 — row 0, col 0
Console.WriteLine(matrix[1, 2]);   // 6 — row 1, col 2
Console.WriteLine(matrix[2, 1]);   // 8 — row 2, col 1

// Dimensions
Console.WriteLine(matrix.GetLength(0));   // 3 rows
Console.WriteLine(matrix.GetLength(1));   // 3 columns

// Iterate
for (int r = 0; r < matrix.GetLength(0); r++)
    for (int c = 0; c < matrix.GetLength(1); c++)
        Console.Write(matrix[r, c] + " ");
```

## 📋 List\<T\> — Dynamic Array

```csharp
var fruits = new List<string> { "Apple", "Banana", "Mango" };

// Add
fruits.Add("Orange");
fruits.AddRange(new[] { "Grape", "Papaya" });
fruits.Insert(1, "Cherry");   // insert at index 1

// Remove
fruits.Remove("Banana");     // by value
fruits.RemoveAt(0);          // by index
fruits.RemoveAll(f => f.StartsWith("P")); // by condition

// Access
Console.WriteLine(fruits[0]);            // first
Console.WriteLine(fruits.Count);         // count
Console.WriteLine(fruits.Contains("Mango")); // True

// Search
int idx   = fruits.IndexOf("Mango");
string?  found = fruits.Find(f => f.Length > 5);

// Sort
fruits.Sort();
fruits.Sort((a, b) => b.CompareTo(a));   // descending

// Convert
string[] arr = fruits.ToArray();
```

## 🗂️ Dictionary\<K, V\> — Key-Value Pairs

```csharp
var scores = new Dictionary<string, int>
{
    ["Asha"]  = 95,
    ["Ravi"]  = 88,
    ["Priya"] = 92,
};

// Add / Update
scores["Kiran"] = 78;    // add new
scores["Asha"]  = 97;    // update existing

// Safe read ✅
if (scores.TryGetValue("Ravi", out int score))
    Console.WriteLine($"Ravi: {score}");   // Ravi: 88

// Check key
bool hasAsha = scores.ContainsKey("Asha");   // True

// Iterate
foreach (var (name, s) in scores)
    Console.WriteLine($"{name}: {s}");

// Keys and Values
List<string> names    = scores.Keys.ToList();
List<int>    allScores = scores.Values.ToList();
```

## 🔵 HashSet\<T\> — Unique Values Only

```csharp
var tags = new HashSet<string> { "csharp", "dotnet", "api" };

tags.Add("csharp");   // already exists — ignored ✅
tags.Add("azure");    // new — added

Console.WriteLine(tags.Count);           // 4
Console.WriteLine(tags.Contains("api")); // True

// Find duplicates in a list
var list = new List<string> { "a", "b", "a", "c", "b" };
var seen = new HashSet<string>();
var dups = list.Where(x => !seen.Add(x)).ToList();
Console.WriteLine(string.Join(", ", dups));   // a, b
```

## 🎫 Queue\<T\> & Stack\<T\>

```csharp
// Queue — FIFO (first in, first out)
var queue = new Queue<string>();
queue.Enqueue("Task 1");
queue.Enqueue("Task 2");
queue.Enqueue("Task 3");
Console.WriteLine(queue.Dequeue());   // Task 1 — first in, first out
Console.WriteLine(queue.Peek());      // Task 2 — look without removing

// Stack — LIFO (last in, first out)
var stack = new Stack<string>();
stack.Push("Action 1");
stack.Push("Action 2");
stack.Push("Action 3");
Console.WriteLine(stack.Pop());   // Action 3 — last in, first out
Console.WriteLine(stack.Peek()); // Action 2
```

## 🌍 Real-World Example: Student Grade Book

```csharp
public class GradeBook
{
    private Dictionary<string, List<int>> _grades = new();

    public void AddGrade(string student, int grade)
    {
        if (!_grades.ContainsKey(student))
            _grades[student] = new List<int>();
        _grades[student].Add(grade);
    }

    public double GetAverage(string student)
    {
        if (!_grades.TryGetValue(student, out var grades))
            return 0;
        return grades.Average();
    }

    public string GetTopStudent()
        => _grades.MaxBy(kvp => kvp.Value.Average())?.Key ?? "None";
}

var book = new GradeBook();
book.AddGrade("Asha",  95); book.AddGrade("Asha",  88);
book.AddGrade("Ravi",  72); book.AddGrade("Ravi",  85);
book.AddGrade("Priya", 99); book.AddGrade("Priya", 96);

Console.WriteLine($"Asha's avg: {book.GetAverage("Asha")}");
Console.WriteLine($"Top student: {book.GetTopStudent()}");
```

## 🎤 Interview Q&A

**Q1: What is the difference between Array and List?**
A: Array = fixed size, fast. List = dynamic size, more methods.
```csharp
int[] arr  = new int[5];     // fixed — can't grow
var   list = new List<int>(); // dynamic — grows ✅
```

**Q2: What is the difference between List and Dictionary?**
A: List = access by index (O(n) search). Dictionary = access by key (O(1) lookup).
```csharp
var list = new List<string> { "Asha" };
var dict = new Dictionary<int, string> { [1] = "Asha" };
Console.WriteLine(dict[1]);   // O(1) — instant ✅
```

**Q3: How do you avoid KeyNotFoundException in Dictionary?**
A: Use `TryGetValue` or `ContainsKey` before accessing.
```csharp
if (dict.TryGetValue("key", out int val))
    Console.WriteLine(val);   // safe ✅
```

**Q4: What is the difference between Queue and Stack?**
A: Queue = FIFO (first in first out). Stack = LIFO (last in first out).
```csharp
Queue<int> q = new(); q.Enqueue(1); q.Enqueue(2); q.Dequeue(); // 1
Stack<int> s = new(); s.Push(1);    s.Push(2);    s.Pop();     // 2
```

**Q5: When would you use HashSet over List?**
A: When you need unique values or fast O(1) `Contains` checks. `List.Contains` is O(n); `HashSet.Contains` is O(1).

## 🚀 Key Takeaways

✅ Arrays — fixed size, fast, zero-indexed  
✅ `List<T>` — dynamic, most common collection  
✅ `Dictionary<K,V>` — key-value, O(1) lookup  
✅ `HashSet<T>` — unique values, O(1) contains  
✅ `Queue<T>` — FIFO · `Stack<T>` — LIFO  
✅ Use `TryGetValue` for safe dictionary access  
✅ Array index starts at 0 · use `[^1]` for last element
